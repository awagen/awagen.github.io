Chapter 1

Kolibri Basics

Kolibri is a clusterable processing framework based on Akka and written in Scala. It is especially focussed on use-cases related to e-commerce search such as search result optimization. The framework is a general one though, allowing distinct types of distributed, clustered processing / state handling.
